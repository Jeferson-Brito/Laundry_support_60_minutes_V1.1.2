Google Chrome Remote

  • Acesse a loja utilizando o Software de acesso remoto AnyDesk.
  • No computador da loja, abra o navegador Google Chrome e digite https://remotedesktop.google.com/ na barra de endereço. Pressione Enter para acessar o site.
  • No site, clique em Acessar o meu computador no canto superior direito da página. Você será redirecionado para uma nova aba.
  • Na nova aba, desça a página até o final e na seção Configurar acesso remoto, clique no ícone de Download para baixar o programa de acesso remoto do Google Chrome.
  • Após o download, execute o arquivo e siga as instruções na tela para instalar o programa.
  • Defina o nome da loja e a senha

    • Lembrando qua a senha deve ser os dois ultimos números da loja + 2010. Exemplo, se a loja for PB05 coloque a senha 052010.

Banco de dados

  • Abra o Valentina Studio 10 digitando seu nome na barra de pesquisa do Windows e pressionando Enter.
  • Conecte-se ao servidor local "PostgreSQL" ou "localhost:5432" com a senha.
  • Exclua o banco de dados antigo "laundry60_totem" e crie um novo com o mesmo nome.
  • Clique com o botão direito no novo banco de dados e escolha "Load Dump".
  • Selecione o arquivo atualizado do banco de dados no HD externo e clique em "Next".
  • Carregue o arquivo no banco de dados e clique em "OK" para finalizar o processo.

Sincronização

  • Para sincronizar, abra o aplicativo "Sistema60minutes.exe" na área de trabalho e mantenha-o aberto por alguns minutos, permitindo a sincronização automática.
  • Para verificar o status da sincronização, clique no ícone de mostrar ícones ocultos (seta para cima) na barra de tarefas e verifique se os ícones dos sistemas da lavanderia estão sincronizados. Se estiverem ainda em processo de sincronização, aguarde até que a operação seja concluída.

Roteador (MikroTik)

  • O cabo de rede está conectado uma extremidade no notebook e a outra no MikroTik, peça para o franqueado conectar a extremidade que está no MikroTik no modem da operadora.
  • Acesse o totem e abra o **Winbox**.
  • Vá em **System** e clique em **Reset Configuration** e aguarde o Winbox reiniciar.
  • Assim que ele reiniciar, o IP Address estará zerado, então você acessa o MAC Address
  • Em seguida, vá em **Files** e exclua todos os arquivos
  • Depois faça o upload do arquivo zipado **MK 60M.zip** e em seguida extraia o arquivo no totem.
  • Em seguida, copie para a pasta **Files**.
  • Após copiar para a pasta Files, vá em **New Terminal** e importe o arquivo **60M.rsc**.
  • Após importar o arquivo volte para **System** e clique em **Reboot**.
  • Pronto, MikroTik configurado/atualizado. Peça para o franqueado voltar o cabo de rede

GET

  • Solicite ao franqueado que efetue a conexão da plaquinha do GET ao totem, utilizando um cabo de dados USB.
  • Uma vez conectada, acesse à loja e faça o envio do arquivo do GET, juntamente com o programa "ESP8266Flasher" que se encontram no drive.
  • Após a transferência dos arquivos, abra o programa "ESP8266Flasher" e acesse a aba "Config". Clique no primeiro ícone de engrenagem e selecione o arquivo específico do GET que foi previamente enviado.
  • Retorne à aba "Operation" e clique no botão "Flash(F)" para iniciar o processo de atualização. Aguarde a conclusão do upload do arquivo.

Crédito em cartão

  • Peça para o franqueado conectar a maquineta normalmente no totem usando o cabo USB.
  • Acesse a loja (totem) através do Chrome Remote e clique com o botão direito do mouse sobre o ícone do windows na barra de tarefas.
  • Selecione a opção Gerenciador de dispositivos e expanda a categoria Portas (COM e LPT).
  • Anote o número da porta COM que corresponde à maquineta, por exemplo COM5.
  • Abra o explorador de arquivos e navegue até a pasta Lavanderia60minutos no disco local (C).
  • Abra o arquivo conf.ini com um editor de texto e localize a linha pinpadport=COM4.
  • Altere o número da porta COM para o que você anotou no passo 4 e salve o arquivo.
  • Em seguida, acesse o sistema Web e navegue até a aba "Lojas" para localizar aquela que requer configuração e clique no botão "Editar". Localize o campo "Código TEF" e copie o código ali disponível. Retorne à loja por meio do Chrome remote e insira o código previamente copiado em "STONECODE=" no arquivo "conf.io" e salve o arquivo.
  • Abra o prompt de comando e digite Microtef restart para reiniciar o serviço da maquineta.
  • Reinicie o totem para garantir a sincronização da maquineta com o totem.
  • Quando o sistema reiniciar, realize um teste para saber se o pagamento em cartão está funcionando normalmente.
